# AMU Course Registration System

Course Registration Portal for Aligarh Muslim University students and faculty.

## Features

- Student Dashboard with CPI tracking
- Course Registration with eligibility checks
- Faculty Dashboard for course management
- Responsive design for mobile and desktop

## Tech Stack

- React + Vite
- TypeScript
- Tailwind CSS
- Axios for API integration

## Getting Started

```bash
npm install
npm run dev
```

## Backend Integration

Update the `baseURL` in `src/services/api.ts` to connect to your backend server.
